# Flutter To-Do List App 🔥

A modern **To-Do List application** built with **Flutter** and **Firebase**.  
This project helped me improve my skills in **Flutter development, Firebase integration, and UI/UX design**.

---

## 🆕 Latest Features & Updates

- Redesigned **UI/UX** for a smoother and cleaner user experience.  
- Implemented **full CRUD operations** (Add, Edit, Delete) for tasks in both **UI and Firebase database**.  
- Added **custom username setup** during first login.  
- Displaying the **user’s name and Google account email** in the profile.  
- Added **confirmation dialogs** before deleting tasks or exiting the app to prevent accidental actions.

---

## ⚡ Technologies Used

- **Flutter** (Dart)  
- **Firebase** (Authentication & Firestore)  
- **Google Sign-In**  
- **Provider / State Management**  
- **Responsive UI for Android & iOS**  

---

## 🔑 Notes

- Make sure you have **Flutter SDK** installed.  
- Firebase setup is required (already configured in the project).  
- Never share your **keystore** or sensitive Firebase credentials publicly.
